# Healthcare-Claims-Denial-Analysis
## Overview
This repository contains an end-to-end analysis of healthcare claims denial patterns. It combines Excel dashboards and Jupyter notebooks to highlight KPIs, denial management strategies, and insurer comparisons.
## Contents
**Health_care_claims_and_analysis_data.xlsx:** Processed claims dataset with pivot tables and dashboards.

**Files_General_Analysis.ipynb**: First-level exploratory data analysis of claims data to identify duplicate rows and unique IDS across each file.

**Merging_analysis_KPIS.ipynb:** Second-level merging multiple files based on the unique IDS.

**KPI_results.ipynb:** KPI creation and adding additional columns for performing calculations.
## Objectives

Identify denial trends across insurers.

Automated comparisons using Excel pivot tables.

Built a dashboard for denial management and revenue analysis.

Create KPIs to measure claims performance.
## Tools & Technologies
**Excel:** Pivot tables, Dashboards, and workflow automation.

**PyCharm Jupyter Notebooks:** Python, Pandas, for analysis and KPI calculation.

**GitHub:** Version control and portfolio presentation.
## How to Use

Download the Excel file and notebooks.

Open the Excel file to view dashboards and pivot tables.
